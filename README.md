# День профийца

Мини-игра (~2–3 мин) про культуру Профи.ру: ценности и «кто такой профиец».

## Как открыть

Открой `index.html` в браузере (двойной клик) или из папки:

```bash
# PowerShell
start index.html
```

Либо локальный сервер:

```bash
npx --yes serve .
```

## Сцены

1. Найм без компромиссов → Сила в людях  
2. Сначала «зачем?» → Здравый смысл  
3. Гипотеза vs презентация → Фокус на результат  
4. Драйв без надрыва  
5. Ошибка без поиска виноватых  

Финал: собранный «профиец» + «что забрать с собой».
